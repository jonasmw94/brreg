# brreg
Oppgave til staccintervju

# Hvordan kjøre program:
 * Naviger inn til innerste brregmappe.
  * Deretter kjører du kommandoen npm install.
 * Når denne kommandoen er utført, skriver du npm start og alt skal funke.
